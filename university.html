Información sobre los proyectos de la asignatura Bases de Datos para Data Warehousing

A lo largo de la asignatura hemos realizado los siguientes proyectos usando postgresql :
- Creación y manipulación de una base de datos relacional.
- Procedimientos almacenados y disparadores.
- Common table expressions y funciones analíticas.

En el proyecto "Creación y manipulación de una base de datos relacional" se pueden ver los scripts que creé para conformar la base de datos

En los proyectos "Procedimientos almacenados y disparadores" y "Common table expressions y funciones analíticas" utilizo la base de datos que creé anteriormente para crear las funciones, disparadores, etc. que me indican los ejercicios.

Los enunciados correspondientes a cada uno de los proyectos son:
Introducción a las bases de datos relacionales
Creación y manipulación de una base de datos relacional
Procedimientos almacenados y disparadores
Common table expressions y funciones analíticas

En los demás documentos de esta carpeta adjunto los scripts que creé en cada proyecto junto con un documento donde se muestran las caturas de pantalla de los resultados y la explicación de los ejercicios teóricos.
